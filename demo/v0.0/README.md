# 核心业务模块首页 Demo

- `index.html`：核心业务模块首页，包含模块切换、统计卡片、快捷筛选、高级筛选和事项列表。
- `detail.html`：三栏式 BPMN 业务事项详情办理页。

使用方式：打开 `index.html`，点击列表中的“查看/办理/修改”即可跳转到 `detail.html`。

说明：本 Demo 为纯前端 mock 数据版本，正式系统中首页列表数据来自后端业务表与自研运行时待办表（`task_instance`）查询，详情页数据来自模块运行时上下文接口、材料附件接口与 RBAC 权限判断接口。Flowable 仅用于 BPMN 解析与流程定义加载，不作为运行时任务源。
